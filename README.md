# ローカルPC用パスワード管理アプリ

ブラウザ上で動作する自分専用のセキュアなパスワード管理ツールです。

## 特徴

- マスターパスワードによる認証
- AES-256-GCMによるデータの暗号化保存
- パスワード自動生成機能
- サービス名・URL・ユーザーID・パスワードのCRUD操作
- データはローカルPC上でのみ管理（外部DBには保存しない）

## 技術スタック

- バックエンド: Go
- フロントエンド: HTML/CSS/JavaScript（バニラJS）
- 暗号化: AES-256-GCM、PBKDF2

## セットアップ

1. 依存関係のインストール
```bash
go mod download
```

2. アプリケーションのビルド
```bash
go build
```

3. サーバーの起動
```bash
./local-sha256gen.exe
```

または

```bash
go run main.go
```

4. ブラウザでアクセス
```
http://localhost:8080
```

## 使用方法

1. 初回起動時、マスターパスワードを設定します
2. マスターパスワードでログインします
3. サービス名、URL、ユーザーID、パスワードを入力して追加します
4. 「生成」ボタンでランダムなパスワードを生成できます
5. パスワード一覧から「表示」「コピー」「削除」が可能です

## データ保存場所

- Windows: `%APPDATA%\local-sha256gen\`
  - `passwords.dat`: 暗号化されたパスワードデータ
  - `master.dat`: マスターパスワードのハッシュとソルト

## セキュリティ

- マスターパスワードはPBKDF2でハッシュ化して保存
- パスワードデータはAES-256-GCMで暗号化
- データはすべてローカルPC上に保存（外部送信なし）
